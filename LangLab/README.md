# LangLab — 每日英日單字學習網站

## 開始使用

用 Chrome 或 Edge 直接開啟 `index.html` 即可，免安裝、免網路（發音需系統語音支援）。

## 功能

- **今日學習**：每天英、日各 10 個高頻日常單字（到期複習優先，新字補滿），含例句、中譯，日文附羅馬拼音。
- **熟悉／不熟悉**：不熟悉的隔天再出現；熟悉的依間隔重複（1→3→7→14→30 天）排程，連續 3 次熟悉即「精通」畢業。
- **測驗**：選擇題＋拼寫題（日文可用羅馬拼音作答），結果計入熟悉進度。
- **發音**：點 🔊 播放單字或例句（英 en-US／日 ja-JP），語速可調。
- **統計**：連續天數、精通數量、近 30 天學習量。
- **設定**：每日字數、深色模式、進度備份匯出／還原。

進度存在瀏覽器 localStorage，**換瀏覽器或電腦前請先到「設定」匯出備份**。

## 擴充字庫

三種方式：

1. **設定 → 新增單字**：手動加一筆。
2. **設定 → 匯入單字**：貼上 JSON，例如：

```json
{"lang":"jp","words":[
 {"word":"約束","romaji":"yakusoku","meaning":"約定",
  "sentence":"約束を守ります。","sentenceRomaji":"yakusoku o mamorimasu.","sentenceZh":"我會遵守約定。"}
]}
```

英文用 `"lang":"en"`，`romaji` 換成 `phonetic`（選填）。

3. **直接編輯 `words.js`**：在陣列中照現有格式追加一行。
   - 日文：`[單字, 羅馬拼音, 中譯, 例句, 例句羅馬拼音, 例句中譯, 等級]`
   - 英文：`[單字, 音標, 中譯, 例句, 例句中譯, 等級]`

內建字庫：英文 300 字（A1–B1）、日文 300 字（N5–N4），依日常使用頻率排序。
